Communicating Delta PLC with WPLSOFT

1.Open WPLSOFT software and click on new file
2.Select PLC model

a.Give programme title
b.Select model type as PLC
c.Select your PLC model(Select proper model or code won't compile) d.Select communication settings.
It can be Ethernet Or RS232

Keep rest settings as it is and click on OK.
Once all setup is done you can write a ladder logic as per your application

Transfering PLC code into PLC( using Ethernet)

After completing ladder logic you need to transfer it to PLC and for that
1.Click on Ethernet
2.Then press auto search ethernet module and you will find IP address of PLC
3.Go to Control Panel ->Network and sharing center
here you will find your plc connection with your pc via ethernet
4.Now click on connection and then click on properties and you will see this screen

5.Now enter the internet protocol version IPV4 and click on manually obtain IP address
6.If IP address of your PLC is say 192.168.1.5 then set your PLC
IP address to same range for eg:(set it to 192.168.1.10)
then subnetmask will be selected automatically.Enter gateway as 192.168.1.1
Now PLC and Your PC are in same range(both have same gateway)
This is only for ETHERNET COMMUNICATION

Transfering PLC code into PLC( using ASCII)

If user wants to use ASCII communication or if user does not have ethernet support
then user can use RS232 & RS485
RS232 is used for transferring code to PLC
RS485 is used for connecting PLC with SCADA applications

For using ASCII communication user needs to install drivers for RS232 & RS485 which
can be downloaded from below
