Validation For Forms

Previously we created a flask login and register.In this tutorial we will study about form validation.Form validation is required to allow users to provide valid information.
This is how the form will look like and if wrong info is provided by user then it will give an alert saying username is invalid or password is invalid

Enter Password:

To achieve such validation we use javascript code

<script type="text/javascript"> function regvalidateform(){ // for storing regemailid and regpasswordid value in variables var regemail=document.getElementById('regemailid').value; var regpassword=document.getElementById('regpasswordid').value; // this is for setting email and password check var emailcheck = /^[A-Za-z_0-9]{3,}@[A-Za-z]{3,}[.]{1}[A-Za-z]{2,6}$/; var passwordcheck = /^(?=.*[0-9])(?=.*[!@#$%^&*])[a-zA-Z_0-9!@#$%^&*]{8,16}$/; if (emailcheck.test(regemail)) { document.getElementById('regusererror').innerHTML = "correct username"; }else { document.getElementById('regusererror').innerHTML = "**username is invalid"; alert("error:username is not valid"); return false; } if (passwordcheck.test(regpassword)) { document.getElementById('regpassworderror').innerHTML = "correct password"; }else { document.getElementById('regpassworderror').innerHTML = "**password is invalid"; alert("error:password is not valid.") return false; } } </script>

When you click on submit button form will call javascript function regvalidateform() which will perform the validation on form. since on form submit we are returning regvalidateform()

onsubmit=" return regvalidateform()"

How regvalidate function works

1. In regvalidate form we are using 2 variables regemail and regpassword, this variables are storing user inputs from register form using their id's

var regemail=document.getElementById('regemailid').value;

var regpassword=document.getElementById('regpasswordid').value;

3. Now 2 more variables are used for providing validation that are emailcheck and passwordcheck.

var emailcheck = /^[A-Za-z_0-9]{3,}@[A-Za-z]{3,}[.]{1}[A-Za-z]{2,6}$/;

var passwordcheck = /^(?=.*[0-9])(?=.*[!@#$%^&*])[a-zA-Z_0-9!@#$%^&*]{8,16}$/;

4. validation starts with /^ and ends with $/;
5.In square brackets you need to provide what you want to include in email address for ex:-1. example01@gmail.com,2.example01@gmail.co.in
6.[A-Za-z_0-9]{3,} so we can have letters from A-Z,letters from a-z , numbers from 0-9 and in curly brackets we need to specify length{min,max} in our case minimum is 3.
7.then you need @ which is compulsory so length is not provided.
8.After @ we need words like gmail,yahoo so in square brackets allow[A-Za-z]{3,}
9.Now . should be compulsory so you can either place it directly like @ or place it in square bracket with length as 1 [.]{1}.
10. At last after . we need to provide com,co,in so allow letters in curly brackets[A-Za-z]{2,6}
11.Some have .co.in so for that you can allow . by using [.]{1} or place . directly and allow letters in square brackets.

12. For passwords we again start with /^ and end with $/;.
13. (?=.*[0-9])
?=.* indicates that your password must include atleast 1 number from 0-9
14. (?=.*[!@#$%^&*])
Similarly for making special characters compulsory in password
15.[a-zA-Z_0-9!@#$%^&*]{8,16}
So in our case password must have one number and one special character and remaining characters can be anything from square bracket and minimum length for password is set as 8 and max as 16.

18. If it does not match then again it will get same span element and replace it with username not valid and also it will show alert as username is invalid.
19. similarly password validation is done
